US Companies Scraper

A simple Python project that scrapes data from Wikipedia about the largest companies in the United States and saves it into a Pandas DataFrame and CSV file.

Features

Scrapes the first table from the Wikipedia page: List of largest companies in the United States by revenue

Extracts company name, revenue, profit, employees, and headquarters

Cleans and organizes the data using Pandas

Saves the data to a CSV file for easy analysis
